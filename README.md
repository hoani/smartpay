## Dependencies

### Cmake
https://cmake.org/install/

### Libmicrohttpd

On Mac:
```bash
brew install libmicrohttpd
```

On Ubuntu:
```bash
sudo apt install libmicrohttpd
```

## Build

### From Command Line

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### VSCode

Install the CMake Tools extension and use the little blue bar at the bottom of the window ;)
